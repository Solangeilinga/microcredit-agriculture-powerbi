# Agricultural Performance & Microcredit Impact Analysis

## Project Overview

This project analyzes agricultural performance and the impact of microcredit programs on farmers in Senegal between 2019 and 2023.

The objective is to evaluate:
- agricultural productivity
- financial performance of microcredit
- regional and crop performance
- repayment behavior of farmers

The analysis was conducted using Power BI.

## Key Insights

- Average yield remained relatively stable between 2019 and 2023
- Some farmers significantly outperform others
- Microcredit repayment rate reaches approximately 78%
- No strong correlation between cultivated area and yield

## Dashboard Features

The dashboard includes:

- Yield evolution over time
- Yield by crop
- Yield by region
- Credit distribution
- Credit repayment analysis
- Farmer performance comparison
